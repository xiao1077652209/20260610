# 模型损失约束（最底部）
默认启用总损失：
总损失 = 带标签平滑0.05主交叉熵损失
        + 0.05×图像分支辅助交叉熵损失
        + 0.10×光谱分支辅助交叉熵损失
可选扩展损失（默认全部关闭）
1. CenterLoss：约束同类融合特征空间紧凑
2. ModalAlignLoss：开启后增加正则项0.01 * mean(1 - cosine(xm, xv))，对齐双模态特征分布

ACGF 自适应通道门控融合模块内部流程
输入 xm(512)、xv(512)
    │
    ├─ xm → image_proj → hm(256)
    └─ xv → spectral_proj → hv(256)
            │
            ├─ interaction = hm * hv （模态乘积交互）
            └─ difference = abs(hm - hv) （模态差异）
                    │
            ┌────────┴────────┐
            ↓                 ↓
      image_gate生成      spectral_gate生成
    输入[hm,interaction,diff]   输入[hv,interaction,diff]
            │                 │
            ↓                 ↓
    加权重构图像特征     加权重构光谱特征
            │                 │
            └────────┬────────┘
                     ↓
            残差细化层（融合细化特征+原始hm、hv）
                     │
                     ↓
             拼接[细化特征, difference]
                     │
                     ↓
             压缩层输出 fused [B,512]

Attn-CNN 光谱分支内部细节
一维输入 [B,1,777]
    │
    ├─ Conv1d(1→128, k=7) → BN → GELU → AvgPool
    │        ↓
    │    光谱注意力层1
    │
    ├─ Conv1d(128→256, k=5) → BN → GELU → AvgPool
    │        ↓
    │    光谱注意力层2
    │
    ├─ Conv1d(256→128, k=3) → BN → GELU → AvgPool
    │        ↓
    │    光谱注意力层3
    │
    ├─ 自适应平均池化 + 自适应最大池化 特征拼接
    │
    └─ Linear+LN+GELU+Dropout → Linear映射至512维 xv

ResNet50 图像分支内部细节
RGB图像输入 [B,3,224,224]
    │
    ↓
预训练ResNet50（移除原始分类头，冻结早期模块：conv1、bn1、layer1）
    │
    ↓
输出2048维视觉特征
    │
    ↓
Dropout(0.1)
    │
    ↓
Linear(2048→512) → L2归一化 → xm [B,512]


深度分类头内部结构
融合特征 fused [B,512]
    │
    ↓
LayerNorm(512) → Linear(512→256)
    │
    ↓
LayerNorm(256) → GELU → Dropout(0.15)
    │
    ↓
Linear(256→类别数) → 输出分类Logits
