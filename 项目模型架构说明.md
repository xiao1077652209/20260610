# 项目分析报告：MFFN-INIRS 产地数据集优化

## 一、项目概述

本项目实现了一个多模态特征融合网络（MFFN-INIRS），用于近红外光谱分类。支持光谱+小波双视图、光谱+图像多模态等多种模式。当前主要针对**产地数据集**（535个样本，7个类别）进行优化。

## 二、两次实验运行结果分析

### 关键发现

#### 1. 严重过拟合
- 训练准确率很快达到 **96-99%**，但验证准确率最高只有 **82-84%**
- 训练损失持续下降，验证损失震荡上升
- 参数/样本比高达 **7,480**（400万参数 / 535样本）

#### 2. 小波辅助损失完全无效
- Aux Loss 从 Epoch 1 到 Epoch 38 始终稳定在 **0.2915 ± 0.0005**
- 原因：小波特征和主光谱特征来自同一输入信号，高度相关
- 单层 Linear 辅助分类头无法学到有效判别信息

#### 3. 类别不平衡
```
类0: 52  类1: 127  类2: 43  类3: 59  类4: 24  类5: 147  类6: 83
```
- 类4只有24个样本，过采样到147个（合成123个），合成样本质量差
- 测试集类4仅3个样本，错1个就是33%误差

#### 4. 小波双视图 vs 仅光谱
- 小波双视图（81.5%）> 仅光谱（77.8%），提升 **+3.7%**
- 说明小波频带信息对产地分类确实有帮助

## 三、优化方案

### 修改1：降低模型容量（models/spectral_extractors.py）

**MultiScaleSpectralCNN 修改：**
- 分支通道数：48 → **32**
- 融合通道数：192 → **128**
- 投影层维度：512 → **256**
- 残差块Dropout：0.08/0.10/0.12 → **0.15/0.20/0.25**
- 投影层Dropout：0.25 → **0.30**

预期参数量从约400万降至约**150-200万**。

### 修改2：增强辅助分类头（models/mffn_inirs.py）

**辅助分类头从单层 Linear 改为两层 MLP：**
```python
# 旧：单层 Linear
self.secondary_aux_classifier = nn.Linear(feature_dim, num_classes)

# 新：两层 MLP
self.secondary_aux_classifier = nn.Sequential(
    nn.Linear(feature_dim, feature_dim // 4),
    nn.LayerNorm(feature_dim // 4),
    nn.GELU(),
    nn.Dropout(0.15),
    nn.Linear(feature_dim // 4, num_classes),
)
```

### 修改3：优化训练配置（mffn_config.py）

| 参数 | 旧值 | 新值 | 说明 |
|------|------|------|------|
| feature_dim | 384 | **256** | 降低特征维度 |
| strategy | max | **median** | 减少合成样本 |
| LEARNING_RATE | 1e-4 | **5e-5** | 降低学习率 |
| WEIGHT_DECAY | 3e-4 | **5e-4** | 增强正则化 |
| FUSION_DROPOUT | 0.10 | **0.20** | 增强正则化 |
| WAVELET_AUX_LOSS_WEIGHT | 0.15 | **0.05** | 降低无效辅助损失 |
| SPECTRAL_AUX_LOSS_WEIGHT | 0.00 | **0.10** | 开启光谱辅助损失 |
| 各分支学习率 | 1e-4 | **5e-5** | 统一降低 |

### 修改4：添加类别权重损失（pipeline.py）

在 `train_model` 函数中添加 `_compute_class_weights`，根据训练集各类别样本数量计算权重，在 CrossEntropyLoss 中使用 `weight` 参数。

## 四、预期效果

1. **降低过拟合**：模型容量降低50%+，正则化增强，参数/样本比从7,480降至约3,000
2. **辅助损失有效**：MLP辅助分类头能学到判别信息，Aux Loss不再恒定
3. **类别平衡**：median策略减少合成样本数量，类别权重让模型关注少数类
4. **训练更稳定**：学习率减半，权重衰减增加，训练过程更平滑

## 五、运行方式

```bash

```
